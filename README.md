# Mesto React

Мой PET проект для практики TypeScript и React

## функциональность

Проект включает в себя:
    Авторизацию
    Работу с API
    Добавление,удаление карточек и лайк карточек


Cтек проекта:
    React
    TypeScript
    React-router-dom v6

## Запустить проект

npm install<br />
npm start
