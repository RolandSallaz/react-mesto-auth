# Mesto React

Место-Реакт это копия проекта Место, переведенного на фреймворк React

## функциональность

проект написан на React , html и css

## Запустить проект

проект написан на React версии 17.0.2

установить все необходимые расширения и запустить проект:
npm install
npm start
